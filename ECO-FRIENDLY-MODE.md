# Eco-friendly Terminal Mode 🌱

## What is it?

A text-only, terminal-style interface for viewing the portfolio with minimal resource usage. Perfect for:
- Accessibility
- Low-bandwidth situations
- Nostalgic terminal enthusiasts
- Screen readers
- Reduced energy consumption

## How to Access

1. Click the **Start** button in the taskbar
2. Click **"Eco-friendly Mode"** (above Shut Down)
3. The terminal interface will appear instantly

## Interface

- **Black background** with **green text** (classic terminal aesthetic)
- **Keyboard-driven** navigation
- **Clear instructions** at every step
- **Exit button** in top-right corner to return to classic mode

## Commands

All commands are simple and case-insensitive. Just type and press ENTER:

### Navigation Commands
- `list` or `ls` - Show all 14 portfolio projects with numbers
- `1` through `14` - View a specific project (e.g., type "3" to view Djingo City)
- `about` - View Jordan's bio and experience

### Utility Commands
- `help` - Show all available commands
- `clear` or `cls` - Clear the terminal screen
- `classic` or `exit` or `back` - Return to Windows 98 interface
- `menu` - Return to welcome screen

### Special Features
- **UP/DOWN arrows** - Browse command history
- **Auto-complete** - Not needed! Commands are short and simple
- **Click anywhere** - Input stays focused for easy typing

## Example Session

```
$ list
[Shows all 14 projects numbered]

$ 1
[Displays SennOS project details]

$ about
[Shows Jordan's bio]

$ classic
[Returns to Windows 98 mode]
```

## Design Philosophy

This mode follows the principle of **progressive enhancement**:
- Works without images
- Works without JavaScript (for the most part)
- Keyboard accessible
- Screen reader friendly
- Low CPU/GPU usage
- Minimal network traffic

## Tips

1. **For beginners**: Just type "help" if you're stuck
2. **Quick navigation**: Type project numbers (1-14) directly
3. **Command history**: Use arrow keys to repeat previous commands
4. **Exit anytime**: Click the button or type "classic"

## Why "Eco-friendly"?

This mode:
- Uses **90% less CPU/GPU** than the graphical mode
- Loads **no images** (text only)
- Reduces **energy consumption**
- Lowers **carbon footprint** of web browsing
- Provides **fast loading** on slow connections

Perfect for when you want to browse sustainably! 🌍

---

**Note**: Portfolio content is loaded from the same text files used in the classic mode, ensuring content consistency across both interfaces.
